# 📄 thesis-writer — Claude Skill

> A natural-language-driven, PhD-level thesis generator for Claude.  
> Describe your research in plain English. Get back publication-ready **LaTeX** or **Markdown** — with auto-TOC, figure/table numbering, Mermaid.js architecture diagrams, BibTeX citations, and inline edit suggestions.

---

## ✨ What It Does

| Feature | Detail |
|---|---|
| **Dual output modes** | LaTeX (`.tex` + `.bib`) for Overleaf/institutions, or Markdown (`.md` + `.bib`) for GitHub/Obsidian/Pandoc |
| **Auto structure** | TOC, List of Figures, List of Tables — all generated by LaTeX/Pandoc, never hand-written |
| **Auto numbering** | Figures, tables, equations, chapters — all cross-referenced with `\ref` / `[@key]` |
| **Mermaid.js diagrams** | Architecture diagrams, flowcharts, pipelines — rendered in-chat AND in the output file |
| **IEEE citations** | Full BibTeX management; converts paper descriptions into `.bib` entries |
| **Edit suggestions** | Paste any paragraph — get a rewrite with tracked changes and rationale |
| **PhD depth** | Lit review synthesises ≥5 sources per theme with "Gap identified" statements; methodology is fully reproducible |
| **No placeholders** | Skill pauses and asks for any missing metadata — never outputs `[Author Name]` |

## 🚀 Quick Start

### Option A — Install in Claude.ai
1. Download [`SKILL.md`](./SKILL.md)
2. Go to **Claude.ai → Settings → Skills → Upload Skill**
3. Upload `SKILL.md`
4. Start a new chat and say: *"Write my thesis on [your topic]"*

### Option B — Use as a System Prompt
Copy the contents of `SKILL.md` and paste it as a system prompt in the API or any Claude-compatible interface.

## 🛠 Compiling the Output

### LaTeX → PDF (via Overleaf — free, no install)
1. Go to [overleaf.com](https://overleaf.com) and create a new project
2. Paste `thesis.tex` as the main file
3. Upload `references.bib`
4. Click **Compile**

### LaTeX → PDF (local)
```bash
pdflatex thesis.tex
bibtex thesis
pdflatex thesis.tex
pdflatex thesis.tex
```

### Markdown → PDF / Word / LaTeX (via Pandoc)
```bash
# PDF
pandoc thesis.md --citeproc --bibliography references.bib -o thesis.pdf

# Word
pandoc thesis.md --citeproc --bibliography references.bib -o thesis.docx

# LaTeX
pandoc thesis.md --citeproc --bibliography references.bib -o thesis.tex
```
